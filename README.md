## Architecture

The project uses MVVM & [Android Architecture Components](https://developer.android.com/topic/architecture).

- UI
  - Activities & Fragments
    - MainActivity
    - MusicActivity
    - SettingsActivity
    - AboutActivity
    - IdentifyFragment
    - HistoryFragment
  - View Models
    - IdentifyViewModel
    - HistoryViewModel
    - SettingsViewModel
- Repositories
  - IdentifyRepository
  - HistoryRepository
  - SettingsRepository
- Data
  - Sources
    - IdentifyDataSource
        - ShazamIdentifyDataSource
        - More in future...
    - HistoryDataSource
  - Models & Entities
    - Music
    - HistoryItem
  - API: Retrofit
    - ShazamAPI
    - ShazamRetrofitInstance
  - DB: Room
    - HistoryItemDao
    - HistoryItemDatabase

### Libraries

- [Coil](https://coil-kt.github.io/coil/)
- [Retrofit](https://square.github.io/retrofit/)
- [Room](https://developer.android.com/training/data-storage/room)
- ...
